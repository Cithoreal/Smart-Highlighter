# firefox-tracker-extension

Group data into browsing sessions with goals and overall sentiment.
Automatically provide data to AI in sidebar
Have data backend with both objective reports from this extension and LLM generated summaries and sentiment analysis from LLM api.
Automatic popups for thought updates or have the LLM prompt in the chat every so often

LLM automatically group tabs based on session or activity.


Install this web extension into Firefox
Other browsers not yet supported

Click on the icon to set your username 

Report your intentions and rate the page you're on
